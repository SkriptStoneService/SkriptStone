And you're probably wondering what’s the difference between Skripts and mods?

Skripts are lightweight server side scripts written in a simplified language using the Skript plugin. They let you add features, mechanics, and commands to a Minecraft server without needing to code in Java or modify the client. They're perfect for servers that want fast, flexible, and easy to manage customizations.

Mods, on the other hand, are more complex and powerful. They’re typically written in Java and require players to install them on their own game clients. Mods can change the game's mechanics at a much deeper level adding new blocks, items, dimensions, and more immersive systems that wouldn’t be possible with Skript alone.

I plan to support both in the future, giving players and server owners more ways to build truly unique experiences.
